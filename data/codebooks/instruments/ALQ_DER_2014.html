Component Description

These are the derived variable(s) for the Alcohol Use (ALQ) self-administered questionniare module.

Analytic Notes

All derived variables in this dataset used variables collected from the self-administered ALQ questionnaire. Sample weights should be used when analyzing these data (see the Analytic Guidelines for information on what weights to use). All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.
