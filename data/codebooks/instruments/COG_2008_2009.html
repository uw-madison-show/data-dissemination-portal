Component Description

The Mini-Cog test is a 3-minute instrument to screen for cognitive impairment. The Mini-Cog uses a three-item recall test for memory and a simply scored clock-drawing test (CDT). The latter serves as an “informative distractor,” helping to clarify scores when the memory recall score is intermediate. 

Eligible Sample

SHOW participants ages 21-74 are eligible for this section.

Interview Setting and Mode of Administration

The COG section is completed during the Time 3 (survey center) visit. It is PAPI administered by trained interviewers.

Data Processing and Editing

The COG section data were systematically reviewed for logical, processing, and data input errors by staff at the Survey of the Health of Wisconsin.

Analytic Notes

These data were collected as part of the Time 3 portion of the SHOW study T3 sample weights should be used when analyzing these data.

 

Between 2008 and 2010, most paper surveys included less detailed response options for COG020 and COG050.  These options included only if all three words were correctly repeated or not. To be consistent with scoring recommendations for the Mini-Cog, coding for these questions changed in 2010. The trained interviewers now write in the total number of correct repeated words.  In this raw 2008-2009 dataset the new coding options include COG020_R1 and COG050_R1, and data were recoded when it was possible (using the hard copy versions of the questionnaires) to determine the actual number of correctly repeated words.

 

References

Borson S. The mini-cog: a cognitive “vitals signs” measure for dementia screening in multi-lingual elderly  Int J Geriatr Psychiatry 2000; 15(11):1021.
