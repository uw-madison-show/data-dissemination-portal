Component Description

The first part of the cognitive instrument includes a common verbal category fluency test in which respondents are asked to name as many different animals as they can in one minute. The Mini-Cog test is a 3-minute instrument to screen for cognitive impairment. The Mini-Cog uses a three-item recall test for memory and a simply scored clock-drawing test (CDT). The latter serves as an “informative distractor,” helping to clarify scores when the memory recall score is intermediate.

Eligible Sample

SHOW participants ages 21-74 are eligible for this section.

Interview Setting and Mode of Administration

The COG section is completed during the Time 3 (survey center) visit. It is PAPI administered by trained interviewers.

Data Processing

The COG section data were systematically reviewed for logical, processing, and data input errors by staff at the Survey of the Health of Wisconsin.

Analytic Notes

These data were collected as part of the Time 3 portion of the SHOW study. T3 sample weights should be used when analyzing these data.

Between 2008 and 2010, most paper surveys included less detailed response options for COG020 and COG050. These options included only if all three words were correctly repeated or not. To be consistent with scoring recommendations for the Mini-Cog, coding for these questions changed in 2010. Since 2010, the trained interviewers now write in the total number of correct repeated words (COG020_R1, COG050_R1). COG020 and COG050 variables are not included in the 2011-2013 datasets since those variables are not informative.

Starting in 2013, the "clock draw" test was used only as a distractor task and not scored, so COG030 and COG040 variables were therefore dropped from the dataset. Scores may still be obtained using saved image scans of each clock draw, available upon request.

Three new variables were added for 2013: COG010_COR is a count of how many animals named in COG010 were considered "correct" (i.e., excluding repeated animals or non-animal words); COG015 indicates whether the list of named animals was transcribed by the interviewer from an audio recording rather than live during the test (a procedure added when possible in 2013 to increase accuracy and completeness of the animal names count); and COG025 indicates whether or not the clock draw test was administered.

Starting in 2013, the list of named animals by each participant (in COG010) was also entered into a database. This information has not been included in the COG dataset but may be available upon request.
