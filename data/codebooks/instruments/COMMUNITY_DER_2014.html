Component Description

These are the derived variables for the Sense of community (COMMUNITY) section.

Analytic Notes

All derived variables in this dataset used variables collected during the self-administered portion of the SHOW study and appropriate weights should be used when analysing these data (see the Analytic Guidelines for information on what weights to use). All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.
