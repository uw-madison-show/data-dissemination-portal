Component Description

These are the derived variable(s) for the Diet (DIQ) module administered during Time 3 (exam center) portion of the SHOW survey.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data.

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

DIQ_GREATLAKE_FISH_EATEN - GREAT LAKES TROUT, SALMON OR CARP EATEN

Variable Name DIQ_GREATLAKE_FISH_EATEN
Variable Format FMT_NUMERIC
Description Total amount eaten during year
Calculation IF DIQ025_A=0 THEN DIQ_GREATLAKE_FISH_EATEN =0;
ELSE IF MISSING(DIQ025_A) THEN DIQ_GREATLAKE_FISH_EATEN=DIQ025_A;
IF DIQ025_B=1 THEN DIQ_GREATLAKE_FISH_EATEN =DIQ025_A*365;
ELSE IF DIQ025_B=2 THEN DIQ_GREATLAKE_FISH_EATEN =DIQ025_A*52;
ELSE IF DIQ025_B=3 THEN DIQ_GREATLAKE_FISH_EATEN =DIQ025_A*12;
ELSE IF DIQ025_B=4 THEN DIQ_GREATLAKE_FISH_EATEN =DIQ025_A;
Target Population Age: 21 – 74

N Mean Median Std Dev Minimum Maximum
106 6.36 2.00 13.83 0.0 104.0
