Subject: Diet

Raw Data Source(s): DIQ

Data File: diq_der_2014.sas7bdat

Component Description

These are the derived variable(s) for the Diet (DIQ) module administered during the in home questionnaire (IHQ) poriton of the SHOW survey.

Analytic Notes

These data were collected as part of the CAPI portion of the SHOW study and sample weights should be used when analyzing these data (see the Analytic Guidelines for information on what weights to use).
All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

Description Total amount eaten during year

else if diq025_b=3 then diq_greatlake_fish_eaten =diq025_a*12;
