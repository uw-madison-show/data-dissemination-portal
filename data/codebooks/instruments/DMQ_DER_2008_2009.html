Component Description

These are the derived variables for the Demographics (DMQ) section.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

Poverty levels were calculated using 2008 and 2009 HHS poverty guidelines for the respective samples. In 2008 the number of people supported by this combined family income (INQ201) was missing for 177 out of the 269 respondents, therefore POVERTY_LEVEL which is a function of household size would be missing also. In order to calculate POVERTY_LEVEL for individuals missing INQ201 we substituted the household screener question of how many people live at this address (SCQ090). For each respondent where INCOME_HH_SIZE was set equal to SCQ090, INCOME_HH_SIZE_IMPUTED was set eqaul to one as a flag. Individual and household income questions were asked as income ranges. The midpoint of the income range was used to calculate the poverty income ratio. POVERTY_150 and POVERTY_200 were based solely on the poverty income ratio. In addition to the poverty income ratio, POVERTY_100 used the follow-up question INQ205, which asks the respondent if her combined family income is above or below a specific income level. The CAPI program for 2009 used the 2008 HHS poverty guidelines in question INQ205.

Fourteen respondents only reported other race and specified Hispanic. For these 14 respondents the number of races reported was set to missing because they did not report a race.
