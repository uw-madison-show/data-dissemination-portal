Component Description

These are the derived variables for the Demographics (DMQ) section.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

Poverty levels were calculated using 2011 HHS poverty guidelines. Individual and household income questions were asked as income ranges. The midpoint of the income range was used to calculate the poverty income ratio. POVERTY_150 and POVERTY_200 were based solely on the poverty income ratio. In addition to the poverty income ratio, POVERTY_100 used the follow-up question INQ205, which asks the respondent if her combined family income is above or below a specific income level. The CAPI program for 2011 used the 2008 HHS poverty guidelines in question INQ205, therefore POVERTY_100 is not necessarily more accurate than the poverty income ratio.

Variable Name POVERTY_100
Variable Format FMT_BELOW_ABOVE
Description Uses reported household income, as well as income verification in INQ205 to determine if subject's household is at or below 100% poverty level.
Calculation IF INCOME_HH_MIN<=POVERTY_LEVEL THEN DO;
IF INQ205 IN (1,2) THEN POVERTY_100=INQ205;
ELSE IF INCOME_HH_MID<=POVERTY_LEVEL THEN POVERTY_100=1;
ELSE POVERTY_100=2; END;
Target Population Age: 21 – 74
