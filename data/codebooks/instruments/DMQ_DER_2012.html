Component Description

These are the derived variables for the Demographics (DMQ) section.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

Poverty levels were calculated using 2012 HHS poverty guidelines. Individual and household income questions were asked as income ranges. The midpoint of the income range was used to calculate the poverty income ratio. POVERTY_150 and POVERTY_200 were based solely on the poverty income ratio. In addition to the poverty income ratio, POVERTY_100 used the follow-up question INQ205, which asks the respondent if her combined family income is above or below a specific income level. The CAPI program for 2012 used the 2008 HHS poverty guidelines in question INQ205, therefore POVERTY_100 is not necessarily more accurate than the poverty income ratio.

Five respondents only reported other race and specified Hispanic. For these five respondents the number of races reported was set to missing because they did not report a race.
