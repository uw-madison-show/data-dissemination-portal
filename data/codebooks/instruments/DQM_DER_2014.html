Raw Data Source(s): DQM

Component Description

Derived variable(s) for the Diet for Minors (DQM) module administered during the In-Home Questionnaire portion of the SHOW survey.

Eligible Sample

SHOW participants between 3 and 17 are eligible for this section.

Analytic Notes

Appropriate weights should be used.

Variable Name DQM_FROZEN_MEAL_PER_YEAR
Variable Format FMT_NUMERIC
Description Total amount eaten during year
Calculation If units in days, multiplied by 365; if units in weeks, multiplied by 52; if units in months, multipled by 12; if units in years, multiplied by 1.

DQM_MEAL_AWAY_PER_YEAR - MEALS AWAY FROM HOME PER YEAR

Variable Name DQM_MEAL_AWAY_PER_YEAR
Variable Format FMT_NUMERIC
Description Total number of school lunches and dining away from home. Scaled to a year interval to make it comparable with adult questions.
Calculation If age >= 3 and age < 12 then do; dqm_meal_away_per_year = sum(dqm030*36, dqm040*52); end; if age >= 12 then do; dqm_meal_away_per_year = sum(dqm130*36, dqm140*52); end;
Target Population Age: 3-17
