Subject: Discrimination

Raw Data Source(s): DSC

Data File: dsc_der_2014.sas7bdat

Component Description

These are the derived variables for the Discrimination section during self-administered questionnaire portion of the SHOW survey.

Analytic Notes

Several derived variables (pertaining to z-scores and categorization based on rank order) created for previous cohorts (prior to 2013) are not included in the current dataset but are available upon request.

All derived variables in this dataset used variables collected during the SAQ portion of the SHOW study and SAQ survey weights should be used when analyzing these data. Details for the calculations will be provided upon request.

References

Hickson, DeMarc A., et al. "The associations of multiple dimensions of discrimination and abdominal fat in African American adults: The Jackson Heart Study." Annals of Behavioral Medicine (2012).

DSC_LIFETIME - TOTAL LIFETIME DISCRIMINATION

Variable Name DSC_LIFETIME
Variable Format FMT_NUMERIC
Description The total number of lifetime discrimination experienced
Calculation     array life_vars dsc010_:;
    dsc_lifetime = 0;
    do over life_vars;
        if life_vars = 1 then dsc_lifetime = dsc_lifetime + 1;
    end;
    if nmiss(of life_vars(*)) = 8 then dsc_lifetime = .;
