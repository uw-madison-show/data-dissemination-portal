Component Description

The Economic Hardship Index (EHI) is calculated based on unemployment rate, proportion dependent population, education, income level, proportion in crowded housing, and proportion in poverty at the census block group, census tract, and county levels. Values are scaled to 0-100 for each dimension using the minimum and maximum values within the state at each geographic level. A composite score is calculated using the scaled values from each dimension with equal weighting, then ranked within each geographic level. A lower rank indicates less economic hardship.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. SAS calculations have not been included in this codebook but are available upon request.
