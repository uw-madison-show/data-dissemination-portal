Raw Data Source(s): EOQ

Component Description

These are the derived variables for the EQ-5D (EOQ) module administered during the Time 1 (home visit) portion of the SHOW survey.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data.

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

EQ_INDEX - EQ INDEX, US WEIGHTS

Variable Name EQ_INDEX
Variable Format FMT_NUMERIC
Description Utility index score based on US population-based EQ-5D preference weights.
Calculation ARRAY XIN{5} EOQ010--EOQ050;
ARRAY XOUT{3,5} ZERO1-ZERO5 ONE1-ONE5 TWO1-TWO5; * DIFFERENCES FROM "FULL HEALTH" (I.E. 1) IN DOMAINS;
IF N(OF XIN{*})=5 THEN DO;
    DO J=1 TO 5; * J IS DOMAIN;
        DO I=1 TO 3; * I IS VALUE;
            IF XIN{J}=I THEN XOUT{I,J}=1;

EQ_STATE - EQ-5D HEALTH STATE DESCRIPTOR

Variable Name EQ_STATE
Variable Format FMT_NUMCAT
Description The 5-digit descriptor of the health state, ranging from 11111 for perfect health to 33333 for the worst possible state.
Calculation IF N(OF XIN{*})=5 THEN DO;
    EQ_STATE=INPUT(COMPRESS(EOQ010||EOQ020||EOQ030||EOQ040||EOQ050),BEST5.);
END; ELSE DO;
    EQ_STATE=.;
END;
