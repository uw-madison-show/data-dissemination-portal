Title: Derived Variables - Geography (GEO_DER)

Years of Content: 2011

First Published: May 2013

Last Revised: April 2015

Subject: Geography

Raw Data Source(s): Household sampling frame

Data File: geo_der_2011.sas7bdat

Component Description

The GEO (Geography) section includes Census geography at the block group level or higher, as well as RUCA and Six State urban rural classification for each respondent's residence.

Analytic Notes

The use of geographic data below the county level is restricted to manuscripts with official approval from the SHOW manuscript committee and appropriate IRB approval.

RUCA_7CAT - RUCA URBAN RURAL 7 CATEGORY CLASSIFICATION

Variable Name RUCA_7CAT
Variable Format FMT_RUCA_7CAT
Description RUCA URBAN RURAL 7 CATEGORY CLASSIFICATION
Calculation IF RUCA_CODE_2000 IN (1.0, 1.1) THEN RUCA_7CAT = 1;
IF RUCA_CODE_2000 IN (2.0, 2.1, 3.0, 4.1, 5.1, 7.1, 8.1, 10.1) THEN RUCA_7CAT = 2;
IF RUCA_CODE_2000 IN (4.0, 4,2) THEN RUCA_7CAT = 3;
IF RUCA_CODE_2000 IN (5.0, 5.2, 6.0, 6.1) THEN RUCA_7CAT = 4;

IF RUCA_CODE_2000 IN (10.0, 10.2, 10.3, 10.4, 10.5, 10.6) THEN RUCA_7CAT = 7;
