Title: Derived Variables - Housing Characteristics (HOQ_DER)

Years of Content: 2012

First Published: May 2013

Last Revised: April 2014

Subject: Housing Characteristics

Raw Data Source(s): HOQ

Data File: hoq_der_2012.sas7bdat

Component Description

These are the derived variables for the Housing Characteristics (HOQ) module administered during the Time 1 (home visit) portion of the SHOW survey.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3.Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

HOQ083_1 - WATER FILTER: BRITA OR OTHER PITCHER WATER FILTER

Variable Name HOQ083_1
Variable Format FMT_YES_NO
Description Selected "Brita or other pitcher water filter" in any of HOQ083_A through HOQ083_F
Calculation ARRAY NEW083 HOQ083_1-HOQ083_6;
R083=0;D083=0;
ARRAY OLD083 HOQ083_A--HOQ083_F;
DO OVER OLD083;
    IF OLD083=2 THEN HOQ083_1=1;
    ELSE IF OLD083=3 THEN HOQ083_2=1;
    ELSE IF OLD083=4 THEN HOQ083_3=1;
    ELSE IF OLD083=5 THEN HOQ083_4=1;
    ELSE IF OLD083=6 THEN HOQ083_5=1;
ELSE IF OLD083=8 THEN HOQ083_6=1;
ELSE IF OLD083=.R THEN R083=1;
ELSE IF OLD083=.D THEN D083=1;
END;
DO OVER NEW083;
IF NEW083^=1 THEN DO;
IF R083=1 THEN NEW083=.R;
ELSE IF D083=1 THEN NEW083=.D;
ELSE IF NEW083=. THEN NEW083=2;
END;
END;
DROP R083 D083;

    R270=0;D270=0;
    ARRAY OLD270 HOQ270_A--HOQ270_G;

        IF OLD270{I} IN (1:6) THEN NEW270{OLD270{I}}=1;
        ELSE IF OLD270{I}=8 THEN NEW270{7}=1;
        ELSE IF OLD270{I}=.R THEN R270=1;
        ELSE IF OLD270{I}=.D THEN D270=1;
    END;
    DO I=1 TO DIM(NEW270);
        IF NEW270{I}^=1 THEN DO;
            IF R270=1 THEN NEW270{I}=.R;
            ELSE IF D270=1 THEN NEW270{I}=.D;
            ELSE IF NEW270{I}=. THEN NEW270{I}=2;
        END;
    END;
    DROP R270 D270 I;
END;
