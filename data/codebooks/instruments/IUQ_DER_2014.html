Title: Derived Variables - Insurance, Access, Utilization (IUQ_DER)

Years of Content: 2014

First Published: July 2015

Last Revised: August 2015

Subject: Insurance Access and Utilization History

Raw Data Source(s): IUQ

Data File: iuq_der_2014.sas7bdat

Component Description

These are the derived variables for the Insurance Access and Utilization (IUQ) section.

Analytic Notes

All derived variables in this dataset used variables collected during the in-home portion of the SHOW study and appropriate weights should be used when analysing these data (see the Analytic Guidelines for information on what weights to use).

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

This year, five questions (INSURANCE_TYPE_R2, IUQ020_MEDICAID_R2, IUQ020_HIRSP_R2, IUQ020_IHS_R2, IUQ020_MIL_R2) questions are slightly different from the questions from previous years since the question (IUQ020_R2) that these variables are derived from did change between years.

INSURANCE_TYPE_R2 - TYPE OF INSURANCE

Variable Name INSURANCE_TYPE_R2
Variable Format FMT_INSURANCE_TYPE
Description  
Calculation INSURANCE_TYPE_R2=0;
IF IUQ010 IN (.R,.D,.) THEN INSURANCE_TYPE_R2=.;
IF IUQ010>0 THEN DO;
ARRAY TEMP IUQ020_R2_A IUQ020_R2_B IUQ020_R2_C IUQ020_R2_D IUQ020_R2_E IUQ020_R2_F IUQ020_R2_G IUQ020_R2_H;
DO OVER TEMP;
IF TEMP=5 THEN INSURANCE_TYPE_R2="1";
IF TEMP IN (1,2,3,6,8,9,10) THEN INSURANCE_TYPE_R2="2";
END;
END;
