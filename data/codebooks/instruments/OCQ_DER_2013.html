Title: Derived Variables: Occupation (OCQ_DER)

Years of Content: 2013

First Published: February 2014

Last Revised: May 2014

Subject: Occupation History and Occupation and Industry Codes from Census 2000

Raw Data Source(s): OCQ

Data File: ocq_der_2013.sas7bdat

Component Description

These are the derived variables for the Occupation Questionnaire (OCQ) which is a part of the in-home personal interview portion of the SHOW study (Time 1). Occupational and Industry Codes are given for current job and usual job (longest held job).

Data Processing

Employees from UW Survey Center, specially trained to use the Alphabetic Index of Industries and Occupations, filled in codes that matched participants work description derived from OCQ. The coder both used hard-copies of the Alphabetic Index as well as a software developed by UWSC staff. The coders worked in pairs and double coded each other's work.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request. Every participant got assigned an industry and an occupation code, from the Census 2000 Industry and Occupation Index, for current/ and longest held job. The assigned code depended on the participant’s description of his/her job. The index itself and further information on it can be found at: http://www.census.gov/people/io/methodology/indexes.html.

OCQ_UNEMPLOYED - UNEMPLOYED BUT LOOKING FOR WORK

Variable Name OCQ_UNEMPLOYED
Variable Format FMT_YES_NO
Description Unemployed but looking for work
Calculation IF OCQ100 = 3 THEN OCQ_UNEMPLOYED = 1; IF OCQ100 = 1 THEN OCQ_UNEMPLOYED = 2; IF OCQ100 = 2 THEN OCQ_UNEMPLOYED = 2; IF OCQ100 = 4 THEN OCQ_UNEMPLOYED = 2;
Target Population Age: 21 – 74 AND report of “[3] Not working but looking for work”
