Component Description

The Physical Activity and Physical Fitness (PAQ) section asks the respondent about physical activites that were done over the past 30 days. The first series of questions are related to transportation, then daily activity, followed by questions regarding physical activities during leisure time.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request. Some participants provided impossible responses to some of the questions in the physical activies questionnaire. These impossible values were re-coded to “missing” values during data cleaning. If a participant reported doing an activity more than 7 days a week, that response was re-coded to “missing”. If a participant reported doing an activity more than 24 hours each time, that response was re-coded to “missing”. When calculating MET_MIN_WEEK, missing values in any MET components (MET_VIG, MET_MOD, MET_TRANS, MET_HOUSEWK) were set to zero assuming individuals who don't know or refused to report their physical activity are probably not doing that activity. If MET_MIN_WEEK value was the same or larger than impossible value (80640 = 8*24*60*7: vigorous activity 7 days 24 hours per week), value was set to missing. However, large values in individual component of total MET_MIN_WEEK (i.e. MET_VIG, MET_MOD) remained as they are in the dataset. While these values are not impossible, it is recommended that researchers review the data and decide how to handle these large values in their analysis.
