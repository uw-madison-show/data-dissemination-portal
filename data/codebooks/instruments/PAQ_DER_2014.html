Component Description

The Physical Activity and Physical Fitness (PAQ) section asks the respondent about physical activites that were done over the past 30 days. The first series of questions are related to transportation, then daily activity, followed by questions regarding physical activities during leisure time.

Analytic Notes

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request. These data were collected as part of the CAPI portion of the SHOW study and sample weights should be used when analyzing these data (see the Analytic Guidelines for information on what weights to use).

MET_VIG_GPAQ - MET_MINUTES SCORE PER WEEK FOR VIGOROUS ACTIVITY

Variable Name MET_VIG_GPAQ
Variable Format FMT_NUMERIC
Description This is the MET_Minutes score per week for vigorous activity. It was created by assigning a value of 8 to each vigorous activity. The frequency of each vigorous activity per week was multiplied by the duration of the activity, and then the product was multiplied by the factor of 8 to obtain the MET_Minutes score per week for that vigorous activity.
Calculation MET_VIG was calculated by summing the total number of minutes/week spent doing vigorous activites from PAQ205, PAQ210, PAQ250, & PAQ255, then multiplying the number of minutes/week by 8 mets/minute. Details on the calculation are available upon request.
Target Population Age: 18+ AND report of “Yes” in PAQ200 and/or "Yes" to PAQ245

MET_MOD_GPAQ - MET_MINUTES SCORE PER WEEK FOR MODERATE ACTIVITY

Variable Name MET_MOD_GPAQ
Variable Format FMT_NUMERIC
Description This is the MET_Minutes score per week for moderate activity. It was created by assigning a value of 4 to each moderate activity. The frequency of each moderate activity per week was multiplied by the duration of the activity, and then the product was multiplied by the factor of 4 to obtain the MET_Minutes score per week for that moderate activity. MET_MOD was then derived by adding all the MET_Minutes score per week for all moderate activities reported by the participant.
Calculation MET_MOD was calculated by summing the total number of minutes/week spent doing moderate activites from PAQ220, PAQ225, PAQ265, & PAQ270, then multiplying the number of minutes/week by 4 mets/minute. Details on the calculation are available upon request.
Target Population Age: 18+ AND report of “Yes” in PAQ215 and/or "Yes" in PAQ260

Description This is the MET_Minutes score per week for transportation activity. It was created by assigning a value of 4 to reported transportation activity. MET_MIN_WEEK_GPAQ - TOTAL MET_MINUTES SCORE PER WEEK

Variable Name MET_MIN_WEEK_GPAQ
Variable Format FMT_NUMERIC
Description This is the total MET_Minutes score per week. It was created as a sum of the MET_Minutes score per week for vigorous, moderate, and transportation activities.
Calculation MET_MIN_WEEK = SUM(OF MET_VIG, MET_MOD,MET_TRANS);

PHYSACTIVE_GPAQ - PHYSICALLY ACTIVE

Variable Name PHYSACTIVE_GPAQ
Variable Format FMT_YES_NO
Description This variable was created from MET_MIN_WEEK to identify physically active participants. A participant who has a MET_MIN_WEEK score of 600 or higher is described as beign physically active (1=Yes). A participant who has a MET_MIN_WEEK score less than 600 is described as not being physically active (2=No).
Calculation PHYSACTIVE_GPAQ=2;
IF MET_MIN_WEEK_GPAQ GE 600 THEN PHYSACTIVE_GPAQ=1;
IF MOD_MIN GE 150 THEN PHYSACTIVE_GPAQ = 1;
IF VIG_MIN GE 75 THEN PHYSACTIVE_GPAQ = 1;
IF MET_MIN_WEEK_GPAQ=. THEN PHYSACTIVE_GPAQ=.;
Target Population Age: 18+
