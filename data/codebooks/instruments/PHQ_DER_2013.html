Component Description

These are the derived variables for the Depression Diagnostic and Severity Measure (PHQ-8) administered during the Time 3 (survey center) portion of the SHOW survey.

Analytic Notes

A minimum of 5 responses were required for scoring the PHQ-8. PHQ_SEVERITY can be used as a screening tool based only on symptom frequency; PHQ_DX requires either depressed mood to be present more than half the days in addition to other symptoms.
All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

PHQ_SCORE - DEPRESSION SCORE

Variable Name PHQ_SCORE
Variable Format FMT_NUMERIC
Description Sum of symptom frequency values for depression symptoms (PHQ010-PHQ080).
Calculation ARRAY PHQS PHQ010--PHQ080; IF PHQ_ANS>=5 THEN DO;     PHQ_SCORE=SUM(OF PHQS(*)); END;
Target Population Age: 21 – 74 and number of symptom questions answered (PHQ_ANS) > 5

PHQ_DX - DEPRESSION DIAGNOSIS

Variable Name PHQ_DX
Variable Format FMT_PHQ_DX
Description Diagnosis of depression with anhedonia (PHQ010) or depressed mood (PHQ020) and at least 2 of 8 symptoms (PHQ010-PHQ080) present "more than half the days" (>=2). Major depression: at least 5 symptoms present "more than half the days" including anhedonia or depressed mood
Calculation ARRAY PHQS PHQ010--PHQ080; IF PHQ_ANS>=5 THEN DO;     IF PHQ010>=2 OR PHQ020>=2 THEN DO;         POS=0;         DO OVER PHQS;             IF PHQS>=2 THEN POS=POS+1;         END;         IF POS>=5 THEN PHQ_DX=1; *MAJOR DEPRESSION;         ELSE IF POS>=2 THEN PHQ_DX=2; *OTHER DEPRESSION; ELSE PHQ_DX=3; *NO DEPRESSION; END; ELSE PHQ_DX=3; *NO DEPRESSION;
Target Population Age: 21 – 74 and number of symptom questions answered (PHQ_ANS) > 5

PHQ_SEVERITY - DEPRESSION SEVERITY

Variable Name PHQ_SEVERITY
Variable Format FMT_PHQ_SEVERITY
Description Screener for depression based on score (PHQ_SCORE). 0-4: no depression; 5-9: mild depression; 10-14: moderate depression; 15-19: moderately severe depression; 20-24: severe depression.
Calculation ARRAY PHQS PHQ010--PHQ080; IF PHQ_ANS>=5 THEN DO;     IF PHQ_SCORE<5 THEN PHQ_SEVERITY=1; *NO DEPRESSION;     ELSE IF PHQ_SCORE<10 THEN PHQ_SEVERITY=2; *MILD DEPRESSION;     ELSE IF PHQ_SCORE<15 THEN PHQ_SEVERITY=3; *MODERATE DEPRESSION; ELSE IF PHQ_SCORE<20 THEN PHQ_SEVERITY=4; *MODERATELY SEVERE DEPRESSION; ELSE PHQ_SEVERITY=5; *SEVERE DEPRESSION; END;
Target Population Age: 21 – 74 and number of symptom questions answered (PHQ_ANS) > 5
