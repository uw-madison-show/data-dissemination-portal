Title: Derived Variables - Depression Screener (PHQ_DER)

Years of Content: 2014

First Published: July 2015

Last Revised: August 2015

Subject: Depression

Raw Data Source(s): PHQ-2

Data File: phq_der_2014.sas7bdat

Component Description

These are the derived variables for the Depression Screener (PHQ-2) administered during the in-home portion of the SHOW survey.

Analytic Notes

Sampling weights should be used when analyzing these data (See 2014 analytic guidelines for more information about which weights to use). All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

References

Kroenke K, Spitzer RL, Williams JB. The Patient Health Questionniare-2: Validity of a Two-item Depression Screener. Medical Care 2003, (41) 1284-1294.

PHQ2_ANS - NUMBER OF QUESTIONS ANSWERED

Variable Name PHQ2_ANS
Variable Format FMT_NUMERIC
Description Number of symptom questions (PHQ010-PHQ020) answered.
Calculation ARRAY PHQS PHQ010 PHQ020; PHQ2_ANS=N(OF PHQS(*)); IF PHQ2_ANS=0 THEN PHQ2_ANS=.;

PHQ2_SCORE - DEPRESSION SCORE

Variable Name PHQ2_SCORE
Variable Format FMT_NUMERIC
Description Sum of symptom frequency values for depression symptoms (PHQ010-PHQ020).
Calculation IF PHQ2_ANS=2 THEN PHQ2_SCORE=SUM(OF PHQ010, PHQ020); ELSE IF PHQ2_ANS<2 THEN PHQ2_SCORE=.;
Target Population Age 18+ and number of symptom questions answered (PHQ2_ANS) = 2

PHQ2_DX - DEPRESSION DIAGNOSIS

Variable Name PHQ2_DX
Variable Format FMT_PHQ2_DX
Description Diagnosis of any depressive disorder.
Calculation IF PHQ2_ANS=2 THEN DO; IF PHQ2_SCORE>=3 THEN PHQ2_DX=1; ELSE IF 0<=PHQ2_SCORE<3 THEN PHQ2_DX=2; ELSE IF PHQ2_SCORE<0 THEN PHQ2_DX=.; END; ELSE IF PHQ2_ANS<2 THEN PHQ2_DX=.;
Target Population Age 18+ and number of symptom questions answered (PHQ2_ANS) =2
