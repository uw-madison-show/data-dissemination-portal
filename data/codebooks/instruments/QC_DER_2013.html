Component Description

These are the derived variables for the Life Events (QC) module included in the Time 2 (self-administered questionnaire) portion of the SHOW survey.

Analytic Notes

In our questionnaire, there are two additional questions (QC_LIFE_44: None of these events happened to me in the past 12 months, QC_LIFE_45: Don't know) at the end of the 43 life events. If a participant marked any event(s) among the first 43 questions AND also marked "none of these events happened to me" or "don't know," we disregarded the information from these last two questions (changing those "yes" responses to "no"). If zero life events were marked but neither QC_LIFE_44 and QC_LIFE_45 was marked instead (i.e., the entire QC section was left blank), then all QC response values were treated as "missing" and SRRS_SCORE was not calculated. QC derived datasets from 2012 and earlier included another variable, SRRS_MISSING, but due to changes in how the data were collected and entered this variable was dropped for 2013 because it is no longer meaningful. (Unless it was clear that the entire QC section was skipped, if there was at least one marked response in QC then any other items left blank/unmarked were assumed to be intentional "No" responses rather than a true "missing" or skipped response because there was no way to distinguish other true missing responses.)

All derived variables in this dataset used variables collected during the Time 2 portion of the SHOW study and T2 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3.Calculation details are available upon request.
