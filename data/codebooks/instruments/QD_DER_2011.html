Title: Derived Variables - QD: Sleep Habits and Problems

Years of Content: 2011

First Published: June 2011

Last Revised: August 2015

Subject: Sleep habits, sleep disorders and excessive daytime sleepiness

Raw Data Source(s): QD_Sleep

Data File: qd_der_2011.sas7bdat

Component Description

These are derived variables from the QD_Sleep section collected during Time 2 of the SHOW study.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 2 portion of the SHOW study and T2 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

SLP_TIMEINBED1 - WORK DAY SELF REPORTED TIME SPENT IN BED FOR CURRENTLY EMPLOYED SP

Variable Name SLP_TIMEINBED1
Variable Format FMT_NUMERIC
Description SLP_TIMEINBED1 was created as the difference between the time the participant usually turned off the lights to go to sleep before a workday or schoolday and the time the participant usually got out of bed to start the day. SLP_TIMEINBED1 was created for participants currently employed or in school, and hours of sleep are reported in decimal format.
Calculation T1=QD_SLP_1C_A+(QD_SLP_1C_B/60);
T2=QD_SLP_1D_A+(QD_SLP_1D_B/60);
T3=(QD_SLP_1C_B/60);
IF QD_SLP_1C_C=QD_SLP_1D_C THEN SLP_TIMEINBED1=ABS(T2-T1);
ELSE SLP_TIMEINBED1=(12-T1)+(T2);
IF QD_SLP_1C_C=1 AND QD_SLP_1D_C=1 AND QD_SLP_1C_A > QD_SLP_1D_A AND QD_SLP_1C_A NOT IN (12)THEN SLP_TIMEINBED1=.;
IF QD_SLP_1C_C=2 AND QD_SLP_1D_C=2 AND QD_SLP_1C_A > QD_SLP_1D_A THEN SLP_TIMEINBED1=.;
IF QD_SLP_1C_C=1 AND QD_SLP_1D_C=1 AND QD_SLP_1C_A > QD_SLP_1D_A AND QD_SLP_1C_A =12 THEN SLP_TIMEINBED1=ABS(T2-T3);
Target Population Age: 21 – 74 and currently work or in school.

N Mean Median Std Dev Minimum Maximum
523 7.73 7.75 1.57 0.0 21.0

SLP_SRHOURSOFSLEEP - SELF REPORTED AVERAGE DAILY HOURS OF SLEEP

Variable Name SLP_SRHOURSOFSLEEP
Variable Format FMT_NUMERIC
Description Participant self-reported average daily hours of sleep.
Calculation SLP_SRHOURSOFSLEEP=.;
IF SLP_EMPLOYED=1 THEN DO;
    IF QD_SLP_1A=1 THEN SLP_SRHOURSOFSLEEP = ((1/7* SLP_SRSLEEPHOURS1)+(6/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
    IF QD_SLP_1A=2 THEN SLP_SRHOURSOFSLEEP = ((2/7* SLP_SRSLEEPHOURS1)+(5/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
    IF QD_SLP_1A=3 THEN SLP_SRHOURSOFSLEEP = ((3/7* SLP_SRSLEEPHOURS1)+(4/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
    IF QD_SLP_1A=4 THEN SLP_SRHOURSOFSLEEP = ((4/7* SLP_SRSLEEPHOURS1)+(3/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
    IF QD_SLP_1A=5 THEN SLP_SRHOURSOFSLEEP = ((5/7* SLP_SRSLEEPHOURS1)+(2/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
    IF QD_SLP_1A=6 THEN SLP_SRHOURSOFSLEEP = ((6/7* SLP_SRSLEEPHOURS1)+(1/7* SLP_SRSLEEPHOURS2)+SLP_HOURSOFNAP);
IF QD_SLP_1A=7 THEN SLP_SRHOURSOFSLEEP = SLP_SRSLEEPHOURS1 + SLP_HOURSOFNAP;
IF SLP_SRSLEEPHOURS1=0 THEN SLP_SRHOURSOFSLEEP=.;
IF QD_SLP_1A<7 AND SLP_SRSLEEPHOURS2=0 THEN SLP_SRHOURSOFSLEEP=.;
IF SLP_SRSLEEPHOURS1=. AND SLP_SRSLEEPHOURS2=. AND SLP_SRSLEEPHOURS3=0 THEN SLP_SRHOURSOFSLEEP=.;
END;
IF SLP_EMPLOYED=2 THEN SLP_SRHOURSOFSLEEP = SLP_SRSLEEPHOURS3+SLP_HOURSOFNAP;
Target Population Age: 21 – 74

N Mean Median Std Dev Minimum Maximum
735 7.16 7.29 1.88 0.0 15.0

SLP_EPWORTH - EPWORTH SLEEPINESS SCORE: (RANGE:0-24)

Variable Name SLP_EPWORTH
Variable Format FMT_NUMERIC
Description SLP_EPWORTH was created as the sum of the responses to the eight Epworth sleepiness scale questions (QD_SLP_23_A -- QD_SLP_23_H), scaled to the number of questions answered. The total score provides an estimate of a person’s propensity for daytime sleepiness in daily life. It has a range of zero (0) to twenty-four (24) with higher scores indicating higher average levels of sleepiness in daily life.
Calculation IF SLP_EPWORTH_ANS > 0 THEN SLP_EPWORTH = (SUM(OF EPWORTH{*}) - SLP_EPWORTH_ANS)*(8/SLP_EPWORTH_ANS);
Target Population Age: 21 – 74

N Mean Median Std Dev Minimum Maximum
