Subject: Discrimination

Raw Data Source(s): QG

Data File: qg_der_2013.sas7bdat

Component Description

These are the derived variables for Section G (Discrimination) during self-administered Time 2 portion of the SHOW survey.

Analytic Notes

Several derived variables (pertaining to z-scores and categorization based on rank order) created for previous cohorts are not included in the 2013 dataset but are available upon request.

All derived variables in this dataset used variables collected during the Time 2 portion of the SHOW study and T2 survey weights should be used when analyzing these data. Details for the calculations will be provided upon request.

References

Hickson, DeMarc A., et al. "The associations of multiple dimensions of discrimination and abdominal fat in African American adults: The Jackson Heart Study." Annals of Behavioral Medicine (2012).

QG_EVERYDAY - TOTAL EVERYDAY DISCRIMINATION

Variable Name QG_EVERYDAY
Variable Format FMT_NUMERIC
Description The total number of everyday discrimination experienced
Calculation ARRAY QG1 QG_DSC_1:;
QG_EVERYDAY=0;
DO OVER QG1;
    IF 0<QG1<7 THEN QG_EVERYDAY=QG_EVERYDAY+1;
END;
IF NMISS(OF QG1(*))=7 THEN CALL MISSING(QG_EVERYDAY);

QG_DSC_DAILYGRP - DAILY DSC CATEGORY

Variable Name QG_DSC_DAILYGRP
Variable Format FMT_QG_3CAT
Description Three categories of daily discrimination
Calculation IF QG_EVERYDAY = 0 THEN QG_DSC_DAILYGRP = 0;
ELSE IF 0 < QG_EVERYDAY < 3 THEN QG_DSC_DAILYGRP = 1;
ELSE IF 2 < QG_EVERYDAY < 8 THEN QG_DSC_DAILYGRP = 2;
