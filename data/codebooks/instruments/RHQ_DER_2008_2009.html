Data File: rhq_der_2008_2009.sas7bdat

Component Description

These are the derived variables for the Reproductive Health (RHQ) section.

Analytic Notes

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data.
