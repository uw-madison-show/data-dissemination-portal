Last Revised: August 2015

Subject: Reproductive Health and Contraception Use

Raw Data Source(s): RHQ

Data File: rhq_der_2014.sas7bdat

Component Description

These are the derived variables for the Reproductive Health Questionnaire, administered during the CAPI-administered portion of the SHOW survey.

Analytic Notes

All derived variables in this dataset used variables collected during the CAPI portion of the SHOW study and sample weights should be used when analyzing these data (see the Analytic Guidelines for information on what weights to use). All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data.

Variable Name HORMONE_USE_DURATION
Variable Format FMT_NUMERIC
Description Time spent taking female hormones, in months.
Calculation hormone_use_duration = .;
    if rhq542_b=1 then hormone_use_duration=rhq542_a/4.3;
    if rhq542_b=2 then hormone_use_duration=rhq542_a;
    if rhq542_b=3 then hormone_use_duration=rhq542_a*12;
