Subject: Prescription medication and OTC drugs history

Raw Data Source(s): RXQ, KEY, HHQ1

Data File: rxq_der_2013.sas7bdat

Component Description

These are the derived variables for the Prescription Medication and Select OTC Drugs which is a part of the in-home personal interview portion of the SHOW study (Time 1).

Analytic Notes

All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. Some of the derived variables (e.g. processed names of medication) which were created in 2008-2010 were not included in this dataset because further data cleaning process is needed for those variables. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

RXQ_ASPIRIN_PRIMARY - TAKING ASPIRIN FOR PRIMARY PREVENTION

Variable Name RXQ_ASPIRIN_PRIMARY
Variable Format FMT_YES_NO_NA
Description Is taking aspirin for primary prevention
Calculation IF (HHQ120=1 OR HHQ130=1 OR HHQ150=1 OR HHQ160=1) THEN RXQ_ASPIRIN_PRIMARY=3; *ASYMPTOMATIC;
ELSE IF (GENDER=1 AND AGE_CONSENT<45) THEN RXQ_ASPIRIN_PRIMARY = 3; *MEN OUT OF AGE RANGE;
ELSE IF (GENDER=2 AND AGE_CONSENT<55) THEN RXQ_ASPIRIN_PRIMARY = 3; *WOMEN OUT OF AGE RANGE;
ELSE RXQ_ASPIRIN_PRIMARY=RXQ296_2;
RUN;
Target Population Adults (men>=45, women>=55 without known stroke and coronary heart disease
