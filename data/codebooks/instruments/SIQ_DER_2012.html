Years of Content: 2012

Subject: Screening and Immunization

Raw Data Source(s): SIQ

Data File: siq_der_2012.sas7bdat

Component Description

These are the derived variables for the Screening and Immunization History which is a part of the in-home personal interview portion of the SHOW study (Time 1).

Analytic Notes

SIQ130_LAST_A1C, SIQ_CHOL_1YR, SIQ_DIAB_3YR variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when analyzing these data. SIQ_BP_2YR, SIQ_BP_1YR variables in this dataset used variables collected during the Time 1 and Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

SIQ130_LAST_A1C - TIME SINCE LAST A1C TEST DATE (MONTH)

Variable Name SIQ130_LAST_A1C
Variable Format FMT_NUMERIC
Description Time since last hemoglobin A1C test
Calculation IF SIQ130_A > 0 THEN SIQ130_C = 15; *Pick middle of the month (15);
SIQ130 = MDY (SIQ130_A, SIQ130_C, SIQ130_B);
SIQ130_LAST_A1C_DAYS = CONSENT_DATE - SIQ130;
Target Population Age: 21-74 and previous report of diagnosis of diabetes in the health history
