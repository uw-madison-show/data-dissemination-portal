Component Description

These are derived variables from the SLM section collected during the CAPI-administered portion of the SHOW study.

Analytic Notes

These data were collected as part of the in-home interview portion of the SHOW study and sample weights should be used when analyzing these data (see the Analytic Guidelines for information on what weights to use). This questionnaire is new in 2014.

SLM_EPWORTH - EPWORTH SLEEPINESS SCORE: (RANGE:0-24)

Variable Name SLM_EPWORTH
Variable Format FMT_NUMERIC
Description SLM_EPWORTH was created as the sum of the responses to the eight modified Epworth Sleepiness scale questions (SLM200--SLM270). The total score provides an estimate of a personâ€™s propensity for daytime sleepiness in daily life. It has a range of zero (0) to twenty-four (24) with higher scores indicating higher average levels of sleepiness in daily life.
Calculation         if slm_epworth_ans > 0 then slm_epworth = (sum(of epworth{*}) - slm_epworth_ans)*(8/slm_epworth_ans);
Target Population Age 12 to 17
