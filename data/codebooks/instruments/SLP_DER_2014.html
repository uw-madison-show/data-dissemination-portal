SLP_EPWORTH - EPWORTH SLEEPINESS SCORE: (RANGE:0-24)

Variable Name SLP_EPWORTH
Variable Format FMT_NUMERIC
Description SLP_EPWORTH was created as the sum of the responses to the eight Epworth sleepiness scale questions (SLP200--SLP270). The total score provides an estimate of a person’s propensity for daytime sleepiness in daily life. It has a range of zero (0) to twenty-four (24) with higher scores indicating higher average levels of sleepiness in daily life.
Calculation if slp_epworth_ans > 0 then slp_epworth = (sum(of epworth{*}) - slp_epworth_ans)*(8/slp_epworth_ans);
Target Population Age 18+
