Subject: Smoking History

Raw Data Source(s): SMQ

Data File: smq_der_2010.sas7bdat

Component Description

These are derived variables about the respondents' smoking history.

References

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

SMQ_DER_FORMER_NEVER_CURRENT - FORMER NEVER CURRENT

Variable Name SMQ_DER_FORMER_NEVER_CURRENT
Variable Format FMT_SMOKER
Description Are you a smoker?
Calculation *SMQ_DER_FORMER_NEVER_CURRENT *never; if smq020=2 then SMQ_DER_FORMER_NEVER_CURRENT=3; else if smq020=.R then SMQ_DER_FORMER_NEVER_CURRENT=.R; else if smq030=.D then SMQ_DER_FORMER_NEVER_CURRENT=.D; *current; else if smq020=1 and smq040=1 then SMQ_DER_FORMER_NEVER_CURRENT=1; *former; else if smq020=1 and smq040=2 then SMQ_DER_FORMER_NEVER_CURRENT=2;

SMQ_DER_PACK_YEAR - PACK YEAR

Variable Name SMQ_DER_PACK_YEAR
Variable Format FMT_NUMERIC
Description A way to measure the amount a person has smoked over a long period of time. It is calculated by multiplying the number of packs of cigarettes smoked per day by the number of years the person has smoked. For example, 1 pack year is equal to smoking 1 pack per day for 1 year.
Calculation *SMQ_DER_PACK_YEAR; if smq040=1 then do SMQ_DER_PACK_YEAR=(smq050/20)*smq060; end; *if former, year_smoking is year_quit-year_start; else if smq040=2 then do years_smoking = smq120 - smq030; SMQ_DER_PACK_YEAR=(smq140/20)* years_smoking; if smq030 = 0 then SMQ_DER_PACK_YEAR = .; *smq030=0 means SP never smoked regulary; end;
Target Population Age: 21 –74
