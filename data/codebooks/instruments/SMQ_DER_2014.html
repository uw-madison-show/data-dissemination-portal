Component Description

These are derived variables about the respondents' smoking history.

Analytic Notes

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request. These data were collected as part of the SAQ portion of the SHOW study and sample weights should be used when analyzing these data (see the analytic guidelines for information on what weights to use).

SMQ_DER_FORMER_NEVER_CURRENT_R2 - FORMER NEVER CURRENT

Variable Name SMQ_DER_FORMER_NEVER_CURRENT_R2
Variable Format FMT_SMOKER
Description Are you a smoker?
Calculation *SMQ_DER_FORMER_NEVER_CURRENT *never; if smq020_R2=2 then SMQ_DER_FORMER_NEVER_CURRENT=3; else if smq020_R2=.R then SMQ_DER_FORMER_NEVER_CURRENT=.R; else if smq030_R2=.D then SMQ_DER_FORMER_NEVER_CURRENT=.D; *current; else if smq020_R2=1 and smq040_R2=1 then SMQ_DER_FORMER_NEVER_CURRENT=1; *former; else if smq020_R2=1 and smq040_R2=2 then SMQ_DER_FORMER_NEVER_CURRENT=2;

SMQ_DER_PACK_YEAR_R2 - PACK YEAR

Variable Name SMQ_DER_PACK_YEAR_R2
Variable Format FMT_NUMERIC
Description A way to measure the amount a person has smoked over a long period of time. It is calculated by multiplying the number of packs of cigarettes smoked per day by the number of years the person has smoked. For example, 1 pack year is equal to smoking 1 pack per day for 1 year.
Calculation *SMQ_DER_PACK_YEAR; if smq040_R2=1 then do SMQ_DER_PACK_YEAR=(smq050_R2/20)*smq060_R2; end; *if former, year_smoking is year_quit-year_start; else if smq040_R2=2 then do years_smoking = smq120_R2 - smq030_R2; SMQ_DER_PACK_YEAR=(smq140_R2/20)* years_smoking; if smq030_R2 = 0 then SMQ_DER_PACK_YEAR = .; *smq030_R2=0 means SP never smoked regulary; end;
Target Population Age: 18 +
