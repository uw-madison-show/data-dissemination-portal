Component Description

These are the derived variables for the spirometry module administered during the Time 3 (survey center) portion of the SHOW survey.

Analytic Notes

Normal population is determined based on self-report of the following conditions: cigarette smoking ever (SMQ020), ever diagnosis of asthma (HHQ210), ever diagnosis of emphysema or chronic bronchitis (HHQ230_R2), persistent cough (HHQ240), or wheezing (HHQ260). Observations are considered valid if 2 readings are within 10% of the maximum reading. Population-predicted values for respondents who reported Hispanic ethnicity were derived from estimates from the Mexican-American population; no population-predicted values were calculated for respondents who indicated a race of "Other".

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3. Shorthand SAS calculations have been included in this codebook for only some variables. For details on the other calculations, see the SPI_DER_CALC file next to the SPI_2011_ANNOTATED file on the website..
