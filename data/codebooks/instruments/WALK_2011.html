Component Description

Walk Score® and Transit Score® results were calculated from the Walk Score website (http://www.walkscore.com) based on respondent addresses.

Analytic Notes

Transit Score results are not available for all addresses. Details of the methodology used to create these scores can be found on the Walk Score website (http://www.walkscore.com). Scores were submitted on May 03, 2012 and are subject to change based on updates to establishments recognized by Walk Score. All derived variables in this dataset used variables collected during the Time 1 portion of the SHOW study and T1 survey weights should be used when ana lyzing these data. All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request.

WALK - WALK SCORE

Variable Name WALK
Variable Format FMT_NUMERIC
English Text Walk score calculated from Walk Score® (http://www.walkscore.com/).
English Instructions The “Street Smart” Walk Score algorithm is based on walking distances from an address to a diverse set of nearby amenities. Certain categories are weighted more heavily than others to reflect destinations associated with more walking trips. In addition, road connectivity metrics such as intersection density and average block length are factored into the score.
