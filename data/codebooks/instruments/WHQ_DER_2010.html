Subject: Weight History

Raw Data Source(s): WHQ

Data File: whq_der_2010.sas7bdat

Component Description

These are the derived variables for the Weight History instrument (WHQ).

Analytic Notes

All derived variables in this dataset used variables collected during the Time 3 portion of the SHOW study and T3 survey weights should be used when analyzing these data. All processing was completed using SAS version 9.3.
