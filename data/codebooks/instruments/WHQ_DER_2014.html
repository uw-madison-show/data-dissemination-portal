Years of Content: 2014

First Published: March 2015

Last Revised: July 2015

Subject: Weight History

Raw Data Source(s): WHQ

Data File: whq_der_2014.sas7bdat

Component Description

These are derived variables for the Weight History Instrument (WHQ), administered during the CAPI-administered portion of the SHOW survey.

Analytic Notes

All processing was completed using SAS version 9.3. Short hand SAS calculations have been included to help researchers recreate these variables from the raw data. Additional details are available upon request. These data were collected as part of the CAPI portion of the SHOW study and sample weights should be used when analyzing these data (see the 2014 Analytic Guidelines for information on what weights to use).
